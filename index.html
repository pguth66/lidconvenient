A Lidconvenient Truth

One of the burning questions of our time is, "Should I put the seat down when I'm done?" This page lets you run simulations to answer this question. There are three strategies, explained below. When you click the button for each strategy, the simulator will run 1000 tests and then show you how many times someone was inconvenienced - which means how often someone had to either put up or put down the seat before using the toilet.

Since the mix of males and females involved can change the results, please enter the number of males and females you'd like to use for the simulation below.
